# webpack-mpa-sample

webpack打包多页面示例项目

## 使用

``` shell
$ git clone https://github.com/Lsnsh/webpack-mpa-sample.git --depth=1
$ cd webpack-mpa-sample
$ npm install
$ npm run dev         # 访问 http://localhost:8080
```
