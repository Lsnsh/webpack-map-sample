# webpack-mpa-sample

Webpack packs a multi-page sample project.

## Usage

``` shell
$ git clone https://github.com/Lsnsh/webpack-mpa-sample.git --depth=1
$ cd webpack-mpa-sample
$ npm install
$ npm run dev         # visit http://localhost:8080
```
